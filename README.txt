Tutorial for creating custom templates for MSE.

The tutorial itself is located in the "Magic Set Editor Template Creation Tutorial.odt" file, 
or can be viewed online at:
https://docs.google.com/document/d/1-jIQAhrqpOrB-E0mNurTGXRkxV1HlA4BgeRFMQMjiIU

This repository also contains the special MSE install referenced throughout the tutorial.

Once you are done reading the tutorial, get the most up to date version of MSE here:
https://github.com/MagicSetEditorPacks

The original repo by Twan van Laarhoven can also be found here:
https://github.com/twanvl/MagicSetEditor2
